# RandomChoice
 Make a list and get an elemenet out randomly.

All items are seperated by ",".

Modes:
    Mode-1:
        Filter items to be used only once.
        Exemple:
            Prefilter = [1,1,2,2,2,2,3,4,5,5]
            Afterfilter = [1,2,3,4,5]
    Mode-2:
        Get multiple output at once.
